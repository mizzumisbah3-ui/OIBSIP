# Unemployment Analysis with Python

## Overview

This project was completed as part of my **Data Science Internship** under the **Data Science Track (Task-2)**. The primary objective was to perform an in-depth analysis of unemployment trends across India using Python and to understand how unemployment patterns changed over time, particularly during the COVID-19 pandemic. By applying exploratory data analysis (EDA) techniques and creating insightful visualizations, this project aims to identify regional differences, monthly trends, and the relationship between key employment indicators.

## Objectives

* Load and inspect the unemployment dataset using Pandas.
* Clean and preprocess the data by handling missing values, renaming columns, and converting data types.
* Perform exploratory data analysis (EDA) to understand the structure and characteristics of the dataset.
* Analyze region-wise average unemployment rates to identify areas with higher unemployment.
* Examine month-wise unemployment trends to observe seasonal and temporal variations.
* Create time-series visualizations for major states to study changes in unemployment over time.
* Identify the top 10 states with the highest average unemployment rates using comparative visualizations.
* Generate a correlation heatmap to examine relationships between unemployment rate, employment, and labour participation rate.
* Compare unemployment rates before and after the COVID-19 outbreak to evaluate the pandemic's impact on employment.

## Technologies Used

* **Programming Language:** Python
* **Libraries:** Pandas, Matplotlib, Seaborn
* **Development Environment:** Jupyter Notebook

## Exploratory Data Analysis

The analysis began with loading the dataset and performing data inspection to understand its dimensions, data types, and missing values. Necessary preprocessing steps, including data cleaning and date conversion, were carried out to prepare the dataset for analysis.

Several visualizations were then created to uncover meaningful insights:

* **Region-wise Analysis:** Compared average unemployment rates across different regions of India.
* **Month-wise Trend Analysis:** Examined how unemployment rates changed throughout the year.
* **Time-Series Analysis:** Visualized unemployment trends over time for selected major states.
* **Top 10 States Analysis:** Identified states with the highest average unemployment rates.
* **Correlation Analysis:** Explored relationships between unemployment rate, employment, and labour participation rate using a heatmap.
* **COVID-19 Impact Analysis:** Compared average unemployment rates before and after the COVID-19 pandemic to understand its economic impact.

## Key Insights

* Unemployment rates varied significantly across different regions of India, highlighting regional economic disparities.
* Some states consistently recorded higher unemployment rates than others throughout the observed period.
* The time-series analysis showed noticeable fluctuations in unemployment, with a significant rise during the COVID-19 pandemic.
* The comparison between pre-COVID and post-COVID periods revealed an increase in average unemployment rates following the nationwide lockdown.
* Correlation analysis provided insights into how unemployment is associated with employment levels and labour participation rates.

## Conclusion

This project provided valuable experience in handling real-world datasets and applying exploratory data analysis techniques using Python. It enhanced my understanding of data preprocessing, statistical analysis, data visualization, and interpreting trends from socio-economic data. Additionally, the project strengthened my practical skills in using Pandas, Matplotlib, and Seaborn to extract meaningful insights and effectively communicate data-driven findings.
