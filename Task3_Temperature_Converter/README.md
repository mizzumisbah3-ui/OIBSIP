# Temperature Converter Website

## Overview

This Temperature Converter Website was developed as part of the **Oasis Infobyte Web Development & Design Internship (Level 1 - Task 3).**

The website is an interactive web application that allows users to convert temperature values between **Celsius, Fahrenheit, and Kelvin**. It features a clean and responsive interface with input validation and user-friendly error handling to ensure accurate conversions.

## Features

* Responsive and User-Friendly Interface
* Convert Temperature Between Celsius, Fahrenheit, and Kelvin
* Numeric Input Validation
* Absolute Zero Validation
* Error Messages for Invalid Input
* Instant Temperature Conversion
* Clean and Centered Layout

## Technologies Used

* HTML5
* CSS3
* JavaScript 

## Learning Outcomes

Through this project, I improved my understanding of:

* JavaScript fundamentals and DOM manipulation
* Temperature conversion logic
* Form validation and error handling
* Creating interactive web applications
* Responsive web design using HTML and CSS
