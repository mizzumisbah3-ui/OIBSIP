# Customer Segmentation Analysis 📊

## Project Overview

This project is completed as part of the **Data Analytics Track - Level 1 (Task 2)**.

The objective of this task is to analyze customer behaviour and segment customers into different groups using the K-Means clustering algorithm. These segments help businesses create personalized marketing strategies and improve customer engagement.  

## Objective

Apply clustering techniques to identify different customer groups based on their digital behaviour, membership duration, and yearly spending patterns.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Project Features

✔ Loaded and explored customer dataset  
✔ Checked missing values and duplicate records  
✔ Performed data cleaning and preprocessing  
✔ Selected important customer behaviour features  
✔ Applied feature scaling using StandardScaler  
✔ Used Elbow Method to find optimal number of clusters  
✔ Implemented K-Means Clustering algorithm  
✔ Visualized customer segments using scatter plots  
✔ Analyzed cluster characteristics  
✔ Generated customer segmentation insights and recommendations  

## Analysis Performed

The following customer attributes were analyzed:

- Time spent on App
- Time spent on Website
- Length of Membership
- Yearly Amount Spent

## Key Insights

- Identified different customer groups based on purchasing behaviour.
- Found high-value customers suitable for loyalty programs.
- Identified low-engagement customers who need targeted promotions.
- Generated strategies for personalized marketing campaigns.

## Business Recommendations

- Provide rewards and premium services to high-value customers.
- Create personalized offers based on customer segments.
- Improve engagement with low-spending customers.
- Use customer segmentation to optimize marketing strategies.


## Conclusion

This project demonstrates practical skills in customer analytics, data preprocessing, clustering algorithms, and extracting actionable business insights from customer data.
