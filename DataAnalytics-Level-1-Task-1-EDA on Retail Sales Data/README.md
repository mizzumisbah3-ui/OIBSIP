# EDA on Retail Sales Data 📊

## Project Overview

This project is completed as part of the **Data Analytics Track - Level 1 (Task 1)**.

The objective of this task is to perform Exploratory Data Analysis (EDA) on a retail sales dataset to uncover patterns, customer behaviour trends, product performance, and actionable business insights.

## Objective

Perform a detailed analysis of retail sales data by applying data cleaning, statistical analysis, and visualization techniques to understand sales trends and customer behaviour.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code

## Analysis Performed

✔ Loaded and inspected the retail sales dataset  
✔ Checked dataset structure, data types, missing values, and duplicates  
✔ Performed descriptive statistical analysis  
✔ Analyzed monthly and quarterly sales trends  
✔ Studied customer demographics including age and gender distribution  
✔ Identified top-selling product categories  
✔ Analyzed revenue contribution by product categories  
✔ Created correlation heatmap for numerical features  
✔ Performed additional customer age group analysis  
✔ Generated business recommendations based on findings  

## Key Insights

- Identified important sales patterns across different time periods.
- Analyzed customer demographics to understand buying behaviour.
- Determined high-performing product categories.
- Extracted meaningful insights to support business decision-making.

## Business Recommendations

- Focus marketing campaigns during high-sales periods.
- Maintain inventory based on product demand.
- Create targeted promotions using customer demographic insights.
- Prioritize high-revenue product categories for business growth.


## Conclusion

This project demonstrates practical data analytics skills including data preprocessing, exploratory analysis, visualization, and deriving actionable insights from real-world retail data.
