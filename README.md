# WanderNest Travels - Landing Page

## Overview

WanderNest Travels is a responsive landing page developed as part of the Oasis Infobyte Web Development & Design Internship (Level 1 - Task 1).

The website is designed for a fictional travel agency and showcases popular destinations, travel services, customer reviews, and contact information through a clean and modern user interface.

## Features

* Responsive Navigation Bar
* Attractive Hero Section
* Popular Destinations Section
* Why Choose Us Section
* Customer Reviews Section
* Contact Information Section
* Mobile-Friendly Design
* Smooth Scrolling Navigation

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries

## Learning Outcomes

Through this project, I improved my understanding of:

* Web page structuring using HTML
* Responsive web design
* CSS styling and layouts
* User interface design principles
