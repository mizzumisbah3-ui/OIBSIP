# Posters & Fliers Design

## Track

**Graphics Design – Task 1**

## Overview

This project was completed as part of the **Oasis Infobyte Graphics Design Internship** under the **Graphics Design Track (Task 1)**.

The objective of this task was to design two creative digital posters using design principles such as visual hierarchy, color harmony, typography, and layout composition.

## Posters

### Poster 1 – FutureTech Summit

A modern technology event poster designed to promote a fictional tech summit featuring AI, innovation, expert speakers, and networking opportunities.

### Poster 2 – CyberCraft

A cybersecurity event poster designed to promote a fictional workshop focused on ethical hacking, cyber awareness, digital security, and hands-on learning.

## Features

* Two unique poster designs
* Defined color palette
* Clear visual hierarchy
* Creative icons and graphic elements
* High-quality PNG export
* Beginner-friendly design approach

## Tools Used

* Canva
* Google Fonts
* Icons & Shapes
* Royalty-Free Images

Through this project, I improved my understanding of:

* Poster layout and composition
* Typography principles
* Color palette selection
* Visual hierarchy
* Creative design techniques
