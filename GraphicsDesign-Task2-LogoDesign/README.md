## Logo Design**

## Overview

This project was completed as part of the **Oasis Infobyte Graphics Design Internship** under the **Graphics Design Track (Task 2).**

The objective of this task was to design a unique and professional logo for a fictional café brand. The logo was created with a focus on simplicity, creativity, and brand identity while maintaining a clean and modern appearance.

## Project

### Café Brand Logo

A modern logo designed for a fictional café brand, representing warmth, quality, and a welcoming atmosphere. The design uses a balanced color palette, clean typography, and simple graphic elements to create a memorable brand identity.

## Features

* Original logo design
* Modern and minimal style
* Balanced color palette
* Scalable design
* Suitable for digital and print media
* Clean and professional branding

## Tools Used

* Canva
* Google Fonts
* Icons & Shapes

## Learning Outcomes

Through this project, I improved my understanding of:

* Logo design principles
* Brand identity creation
* Typography
* Color selection
* Minimal and modern design concepts
