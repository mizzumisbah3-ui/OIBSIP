# Personal Portfolio Website

## Overview

This Personal Portfolio Website was developed as part of the **Oasis Infobyte Web Development & Design Internship (Level 1 - Task 2).**

The website is designed to showcase my profile, technical skills, projects, and contact information through a clean, modern, and responsive user interface. It serves as a personal portfolio to highlight my educational background and web development journey.

## Features

* Responsive Navigation Bar
* Professional Hero Section
* About Me Section
* Skills Showcase
* Projects Section
* Contact Information Section
* Responsive Layout
* Smooth Scrolling Navigation
* Hover Effects for Better User Experience

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries

## Learning Outcomes

Through this project, I improved my understanding of:

* Creating a responsive personal portfolio
* Structuring web pages using HTML
* Styling layouts with CSS
* Improving user interface and user experience
